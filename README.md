# BeautyParlour

## Project Overview
BeautyParlour is a web project for showcasing beauty parlour services and enabling customers to connect through contact and booking flows. The project is organized as a frontend + backend application:

- **Frontend:** HTML, CSS, JavaScript, static assets
- **Backend:** Node.js with Express-style routing/controllers for contact and booking APIs
